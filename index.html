ABOUT ME

We can only see a short distance ahead, but we can see plenty there that needs to be done. - Alain Turing

My name is Sabrina D’Mello. I am currently completing my final semester in Computer Science at Concordia University. I am in the Computer Games Options and I am passionate at programing and learning new techniques to further improve my skills in coding. I can code in: Java, C++, HTML/CSS/Javascript/Php, C#, SQL. I have done many projects in my 3 years of University such as creating a website, recreating a board game with C++ using different design patterns, a procedurally generated graphic project using OpenGL and more recently short games using Unity.

MY PROJECTS

Here are some of the project I have worked on thorughout my years at Concordia

Power Grid

WINTER 2016: This was a project that we worked on in the Advance Program Design with C++ Course. We had to recreate the board game Power Grid. The goal of this project was to use design patterns, such as MVC and Factory, to keep the code organized and well structured.

Procedurally Generated Archipelago

FALL 2016: This was a project that we worked on in the Graphic Design Course. The task for this project was to use OpenGL to program a procedurally generated world. Our team created a procedurally generated Archipelago.

Puzzle Bobble 3D

FALL 2017: This was a Game assignment for the Introduction to Game Development Course. For this assignment we had to recreate the Puzzle Bobble game in 3D.

Song In Motion

FALL 2017: Final Project for the Introduction to Game Development Course. Song in Motion is a 2D metroidvania side scroller game in which the player’s choice of game music affects the in-game abilities at their disposal. The cyber punk theme will give the players a rich environment to affect with the music and powers they choose to wield. Players of Song in Motion will find that by paying attention to the qualities of the musical themes, they may better understand how to use them to solve the challenges of the game.

Bonzee

Fall 2017: Bonzee is a 2 player adversary game on a 9x5 chess-like board. Players are either a red or green token and play against eachother, or they can play versus an AI. The AI uses a MinMax algorithm to try and beat the player.

The Plight of Knights and Sight

Ubisoft Game Lab 2018: The Plight of Knights and Sight is a co-op networked puzzle solving game where both players have the same mechanics and abilities. Each player has different visibilities of obstacles (some obsticals are visible for one player and the same obstacles invisible for the other). Players will need to communicate together in-game by the use of the interactive elements. Using the puzzle pieces such as pullable and pickupable boxes, you will guide each other through the levels.

Mimicry

Winter 2018: Mimicry is a two player networked game, where one player is a mimic and the other is a farmer. The mimic player disguises himself as the NPC animals around the farm to kill them while avoiding being caught by the farmer. While on the contrary, the farmers job is to make sure his animals are healthy and to hunt down the mimic.
